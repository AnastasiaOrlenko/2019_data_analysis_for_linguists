Как выполнять домашние задания?

Домашние задания у всех общие, однако для каждого студента каждый датасет несколько отличается. Все датасеты собраны в этой папке, где содержатся подпапки для каждого студента (если, Вы не видете Вашей папки, заполните эту форму и напишите мне).

Домашние задания будут проверятся полуавтоматически, так что пожалуйста:

  • создайте репозиторий 2019_hse_data_analysis (или переименуйте уже существующий);
  • добавьте в соответствующую папку файл с решением в формате hw, номер домашнего задания, .Rmd;
  • следуйте паттерну называния подпапок повторяющему схему в Вашей папке в репозитории курса;
